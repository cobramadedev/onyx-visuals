# Onyx Visuals

Next.js + Supabase portfolio website.

## Setup

1. `npm install`
2. Copy `.env.local.example` to `.env.local` and fill in your Supabase credentials
3. `npm run dev`

## Deploy to Vercel

1. Push to GitHub
2. Import repo on vercel.com
3. Add environment variables in Vercel dashboard
4. Deploy
